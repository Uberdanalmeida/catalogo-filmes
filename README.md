# 🎬🍿 Catálogo de Filmes

![Catálogo de Filmes](Imagens/Catalogo-de-filmes.png)

Um projeto simples e elegante desenvolvido com **HTML, CSS e JavaScript puro**, que apresenta um **catálogo interativo com os 10 filmes de maior bilheteria da história**.

O usuário pode pesquisar filmes pelo nome e visualizar informações como **ano**, **gênero**, **sinopse** e um **link direto para o IMDb**.

---

## 🖼️ Preview

Interface com tema escuro inspirada em plataformas de streaming, oferecendo uma experiência moderna e agradável.

---

## 📱 Responsividade

O layout se adapta automaticamente a diferentes telas:

💻 Desktop: cards centralizados

📱 Mobile: cards em layout horizontal, com sinopse reduzida

---

## 🚀 Funcionalidades

✅ Listagem dinâmica de filmes em cards  
🔍 Busca por nome do filme  
🎨 Tema escuro estilo Netflix  
📱 Layout totalmente responsivo  
🖼️ Fallback automático para imagens ausentes  
🔗 Link exter
